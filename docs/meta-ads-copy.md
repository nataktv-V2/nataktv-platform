## A/B Testing Notes

- **Test 1:** Hindi vs English copy (same creative) to measure CTR difference
- **Test 2:** ₹2 trial hook vs content-led messaging for conversion rate
- **Test 3:** Feed placement vs Stories placement for CPC
- **Test 4:** Regional language targeting vs broad Hindi belt
- Start with ₹500/day budget, scale winners after 3-5 days of data
- Kill ad sets below 1% CTR after 1000 impressions
- Optimize for "Sign Up" or custom event once Pixel is installed

## UTM Parameters

Use these UTM parameters for tracking:
```
?utm_source=meta&utm_medium=paid&utm_campaign={campaign_name}&utm_content={ad_set_name}&utm_term={ad_name}
```

Example:
```
https://nataktv.com/home?utm_source=meta&utm_medium=paid&utm_campaign=trial_conversion&utm_content=hindi_trial_hook&utm_term=trial_feed_1080
```
